# EXIT IRR Simulator — V2 (Liquid Glass)

Premium **Liquid Glass × Claude Warm** redesign of the [EXCEED EXIT IRR Simulator](https://exceed-realestate.github.io/IRR-simulator/).

Same calculations, same brochures, same JS logic — fully reskinned with:

- Apple-style **liquid glass** panels (backdrop blur + saturated frost + specular highlights)
- A **warm, cream / coral palette** inspired by Claude's design language
- Soft layered shadows, pill-shaped controls, rounded 14–22px radii
- Light-mode-only (the v1 navy / gold theme is gone)

## Features (unchanged from v1)
- 8 Dubai property presets + custom property input
- Off-plan / Ready / Secondary purchase modes with quarterly construction & lease modeling
- Mortgage, FX (HKD/USD/JPY/EUR/CNY/KRW/INR/RUB), appreciation scenarios
- Live IRR, Cash-on-Cash, equity multiple, scenario comparison
- Brochure modal (PDFs in `/brochures/`)
- 30-second guided demo tour
- PDF export of the simulator and the IRR breakdown
- Custom investment report generator
- 日本語 / English language toggle
